Replace `<>` with the Logz.io Listener URL [for your region](https://docs.logz.io/docs/user-guide/admin/hosting-regions/account-region/#available-regions), configured to use port **8052** for http traffic, or port **8053** for https traffic. For example, `listener.logz.io` if your account is hosted on AWS US East, or `listener-nl.logz.io` if hosted on Azure West Europe.
